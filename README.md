## Configuração com GitHub
1. Inicialize o repositório local:

```bash
cd "c:/Users/Alefe/OneDrive/Área de Trabalho/projeto"
git init
```

2. Configure o nome de usuário e e-mail se ainda não o fez:

```bash
git config --global user.name "Seu Nome"
git config --global user.email "seu@email"
```

3. Adicione e faça o commit inicial:

```bash
git add .
git commit -m "Initial commit"
```

4. No GitHub, crie um repositório **público** ou **privado** chamado `ORDEGF2026` (marque como "Private").
5. Siga as instruções fornecidas pelo GitHub para conectar o remoto, por exemplo:

```bash
git remote add origin https://github.com/SEU_USUARIO/ORDEGF2026.git
# ou use SSH se preferir
# git remote add origin git@github.com:SEU_USUARIO/ORDEGF2026.git

git branch -M main

git push -u origin main
```

Isso enviará seu projeto ao repositório privado no GitHub.`
